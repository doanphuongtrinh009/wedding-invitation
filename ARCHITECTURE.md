## Scalability Pattern

```
To add a new feature:

1. Create feature directory
   features/new-feature/

2. Add component
   features/new-feature/NewFeature.tsx

3. Import in page.tsx
   import { NewFeature } from './features/new-feature/NewFeature'

4. Add to page
   <NewFeature />

5. Add navigation link (optional)
   { href: "#new-feature", label: "New Feature" }
```
